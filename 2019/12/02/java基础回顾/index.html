一、数据类型

1. 1 标识符 关键字

  • 标识符
    • 标识符可以由字母、数字、下划线（_）、美元符（$）组成，但不能包含 @、%、空格等其它特殊字符。
      • 不能以数字开头
      • 标识符是严格区分大小写的。
      • 标识符的命名最好能反映出其作用，做到见名知意。
  • 关键字
    • java语言中已经被赋予特定意义的一些单词。一共有53个关键字。其中有两个保留字:const和goto。关键字不能被用作标识符！

1.2 基本数据类型

类型名称 字节空间 范围
byte 1 -27到27-1 或者 -128到127
short 2 -215到215-1
int 4 -231到231-1
long 8 -263到263-1
float 4 单精度，对小数部分的精度要求不高
double 8 双精度，精确的小数部分并操作值很大时
char 2 0到65535
boolean 1 真true 假false

1.3 基本数据类型的字面值

  • 整数字面值是int类型
  • byte,short,char三种比int小的整数可以用范围内的的值直接赋值
  • 浮点数的字面值是double类型
  • 字面值后缀LFD
  • 进制前缀（0x,0,\u）

1.4 基本类型的类型转换

​ byte—>short—>int—>long—>float—>double

  • 小到大/隐式转换，直接赋值
  • 大到小/显式转换，需要强制类型转换
  • 浮点数转成整数，小数直接舍弃

1.5 运算规则

  • 计算结果的数据类型，与最大类型一致

  • byte,short,char三种比int小的整数，运算时会先自动转换成int

1.6 运算符

  • 算数运算符：+ - * / % ++ –
  • 比较运算符：> < == >= <= !=
  • 逻辑运算符： && || ! & |
  • 三元运算符：？ ： 三项运算1 ? 2 : 3，1是真取2，1是假取3
  • 赋值运算符：= += -= *= /=

2.4 continue 和 break

  • break 和 continue 都可以用来控制循环结构，区别是：
    • break 用于中断当前循环，即当前循环完全结束。
    • continue 用于结束本次循环，即跳过 continue 后面的代码，继续执行下一次循环。
    • break 还可以用在 switch case 语句中，用于中断 switch case 语句。
